# Audio System

PanflowXR generates all ambient sound and reverb **entirely in code** — no audio files are loaded. Everything runs through the Web Audio API, independent of IWSDK's audio subsystem.

---

## Architecture

```
AmbientSoundManager
  └─ out (GainNode, controls volume)
       └─ ReverbManager.masterIn
            ├─ dryGain  ──────────────────→ AudioContext.destination
            └─ convolver → wetGain ───────→ AudioContext.destination
```

The ambient synth feeds into the reverb chain, so the reverb setting affects all ambient sounds uniformly.

---

## ReverbManager (`src/reverb.ts`)

Implements a **convolution reverb** using a synthetically generated impulse response (IR).

### Impulse Response generation

```typescript
private _generateIR(duration: number, decay: number): AudioBuffer
```

Generates a stereo buffer of white noise shaped by an exponential decay curve:

```
sample[i] = whiteNoise * (1 - i/length)^decay
```

- `duration` — IR length in seconds (3.0 s)
- `decay` — how fast the tail fades (4.0 = relatively tight)

This mimics how a real room's energy decays after an impulse — the convolver then applies that shape to any audio passing through it.

### Dry/wet mix

```typescript
setWet(t: number)   // 0 = fully dry, 1 = fully wet
```

- Dry gain: `1 - t`
- Wet gain: `t × 3` (amplified so the effect is clearly audible at low settings)
- Both transitions use `setTargetAtTime` with a 100 ms time constant — no clicks.

### Reverb presets (UI)

| Preset     | Wet mix |
|------------|---------|
| Room       | 20%     |
| Hall       | 40%     |
| Cathedral  | 65%     |
| Cavern     | 90%     |

## AmbientSoundManager (`src/ambient.ts`)

Generates looping ambient textures procedurally. Each preset uses a different combination of **noise type**, **filters**, and **LFOs**.

### Signal primitives

**White noise** — equal energy at all frequencies, generated by filling a buffer with `Math.random() * 2 - 1`. Sounds sharp/harsh on its own; shaped by filters into usable textures.

**Pink noise** — energy decreases 3 dB per octave, matching how many natural sounds are distributed. Generated using Paul Kellet's 7-coefficient filter applied to white noise:

```typescript
b0 = 0.99886 * b0 + w * 0.0555179;
b1 = 0.99332 * b1 + w * 0.0750759;
// ... (7 poles)
data[i] = (b0 + b1 + b2 + b3 + b4 + b5 + b6 + w * 0.5362) * 0.11;
```

Result: a warmer, more organic texture than white noise.

**LFO (Low Frequency Oscillator)** — an `OscillatorNode` running well below the audible range (< 1 Hz) used to modulate other parameters over time rather than produce sound directly.

---

### Presets

#### Rain

```
whiteNoise → LowPassFilter(1800 Hz, Q=0.4) → gain(0.55) → out
```

White noise low-passed at 1800 Hz cuts the high-frequency hiss, leaving the soft wash of rainfall.

#### Forest

```
pinkNoise → LowPassFilter(3000 Hz) → gain(0.45) → out
```

Pink noise already has a gentler high-frequency distribution. Light low-pass smoothing produces a soft, leafy texture without harsh edges.

#### Ocean

```
whiteNoise → LowPassFilter(500 Hz) → masterGain → out
                                          ↑
                    LFO(0.12 Hz) → lfoGain(0.35) ──┘
```

Low-passing at 500 Hz isolates the deep rumble. The LFO runs at 0.12 Hz (~8-second cycle) and modulates the master gain — this creates the rhythmic swell and retreat of waves.

#### Wind

```
whiteNoise → BandPassFilter(700 Hz, Q=0.6) → gain(0.4) → out
                    ↑ frequency
      LFO(0.07 Hz) → lfoGain(±300 Hz sweep)
```

A bandpass filter isolates a narrow frequency band, producing a breathy whistle. The LFO sweeps the filter's center frequency ±300 Hz at 0.07 Hz (~14-second cycle), creating the gusting, rising-and-falling character of wind.

## Initialization

Both managers are singletons exported from their modules:

```typescript
export const reverbManager  = new ReverbManager();
export const ambientManager = new AmbientSoundManager();
```

`ReverbManager.init()` creates the `AudioContext` (requires a user gesture — called on first XR session entry). `AmbientSoundManager.init()` receives the context and a `connectFn` to route its output into the reverb chain:

```typescript
reverbManager.init();
ambientManager.init(reverbManager.audioContext, (node) => {
  reverbManager.connectSource(node);
});
```
